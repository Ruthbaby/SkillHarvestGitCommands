What is Git and Github

Git is a version control system that manages and keeps track of your code.
while Github is a service that let you host, share and manage your code files on the internet.
Github uses Git underneath and lets you manage your Git repositories or folders easily on its platform.
Before you start to use github you must create an account

Git Commands

  1. To set your Git username type: git config --global user.name Ruth
  2. To confirm your username type: git config --global user.name
  3. To set your git username type: git config --global user.email

Github Commands

  1. To initialize file type: git init
  2. To add a file type: git add or git add .
  3. To add a file type: git add or git add --all.
  4. To add a file type: git add or git add -A
  5. To commit a file type: git commit -m "first about the file
  6. To know the branch where your file is located type: git branch -M main or Master.
  7. To connect the local folder to the repository on GitHub followed by the repo's url: git remote add origin, then email url
  8. To push your code type: git push -u origin main
  9. To add a ReadMe file type: git add README.md
  10. To Remove a file type: git rm --cached
  11. To check the file status type: git status
  12. To create a local copy of a project that already exist remotely type: git clone
  13. To merge lines of development together and combine changes from two distinct branches type: git merge.
  14. To update the local line of development with updates from its remote counterpart type: git pull
  15. To move or rename a file or a directory type: git mv
